About ResumeRadar

A skill-matching platform built with Flask and MongoDB

What is ResumeRadar?

ResumeRadar is a web-based platform that helps recruiters find the most relevant candidates for their job openings. It uses keyword matching and text comparison to rank applicants based on how well their resumes match job descriptions.

How It Works

Recruiters post job descriptions. Job seekers upload their resumes. The platform compares resumes against job descriptions using TF-IDF vectorization and cosine similarity — techniques commonly used in search engines and text analysis.

Each applicant receives a match score that helps recruiters quickly identify the most promising candidates.

Project Background

ResumeRadar was developed as a Computer Science final year project. The goal was to build a complete full-stack web application that solves a real-world problem: the time-consuming process of manually screening resumes.

The project demonstrates skills in:

  • Backend Development — Flask (Python), MongoDB Atlas
  • Frontend Development — HTML, CSS, JavaScript
  • Deployment — Render cloud platform
  • Authentication — Google OAuth integration
  • Text Processing — TF-IDF vectorization, cosine similarity

Platform Features

For Recruiters

Post jobs, view ranked candidates, shortlist, hire. Dashboard shows all applications with match scores.

For Job Seekers

Upload resumes, browse jobs, see match scores before applying, track applications in real time.

Resume Parsing

Extracts text from PDF and TXT files automatically. Skills, experience and education are identified from resume content.
